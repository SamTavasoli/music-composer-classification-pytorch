# Music Composer Classification Using Deep Learning (PyTorch)

## Overview

This project uses deep learning to classify classical music composers from MIDI files.

The project compares two PyTorch models:

- Long Short-Term Memory (LSTM)
- Convolutional Neural Network (CNN)

The composers included are:

- Bach
- Beethoven
- Chopin
- Mozart

## Project Structure

- data/ - Dataset
- notebooks/ - Jupyter notebooks
- src/ - Python source code
- models/ - Saved trained models
- reports/ - Final project report
