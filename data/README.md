# Dataset Instructions

This project uses the Kaggle dataset:

**midi_classic_music**  
Source: Kaggle  
URL: https://www.kaggle.com/datasets/blanderbuss/midi-classic-music

## Dataset Description

The dataset contains MIDI files of classical music works from many composers.  
For this project, only four composers are used:

- Bach
- Beethoven
- Chopin
- Mozart
